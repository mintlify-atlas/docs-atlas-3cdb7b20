UABEA (Unity Asset Bundle Extractor Avalonia) is a cross-platform tool for reading and writing Unity asset bundles and serialized files. Originally inspired by the classic UABE tool, UABEA is built from the ground up for modern Unity versions and runs on Windows, Linux, and macOS.

<Note>
  UABEA is primarily a **modding and research tool**. If you only need to extract assets for viewing, consider [AssetRipper](https://github.com/AssetRipper/AssetRipper) or [AssetStudio](https://github.com/Perfare/AssetStudio/) instead.
</Note>

## Get started

<Steps>
  <Step title="Download UABEA">
    Grab the latest nightly build for [Windows](https://nightly.link/nesrak1/UABEA/workflows/dotnet-desktop/master/uabea-windows.zip) or [Linux](https://nightly.link/nesrak1/UABEA/workflows/dotnet-ubuntu/master/uabea-ubuntu.zip), or download a stable release from the [GitHub Releases page](https://github.com/nesrak1/UABEA/releases).
  </Step>
  <Step title="Open a file">
    Launch UABEA, then use **File → Open** to load an asset bundle or serialized assets file from your game directory.
  </Step>
  <Step title="Browse and edit assets">
    Select assets in the list, click **View Data** to inspect fields, or use **Export Dump** / **Import Dump** to edit via JSON.
  </Step>
  <Step title="Save your changes">
    Use **File → Save** or **File → Save As** to write your changes back to disk. Create an EMIP package to share your mod.
  </Step>
</Steps>
